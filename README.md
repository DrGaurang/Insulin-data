# Analysis of Insulin protein for various organisms

## Objective
To analyze Insulin Protein across species using alignment and phylogenetic analysis.

## Tools Used
- Uniprot (data retrieval)
- Clustal Omega (alignment and Phylogenetic tree)

## Organisms
Homo sapiens	=Human
Sus scrofa	=Pig
Bos taurus	=Cow
Felis catus	=Domestic cat
Octodon degus	=Degu
Macaca mulatta	=Monkey
Gallus gallus	=Chicken
Cavia porcellus	=Guinea pig

## Workflow
1. Retrieved Insulin Protein sequences from Uniprot
2. Conducted multiple sequence alignment
3. Built phylogenetic tree
